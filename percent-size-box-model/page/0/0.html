Задайте вложенному блоку размеры width и height в %
соответствующие комментариям, указанным в px. 

          .parent {
            width:  600px;
            height: 300px;
          }
          .child {
            width: %;  // 420px
            height: %;  // 210px
          }
Следующее задание
.parent { width: 600px; height: 300px}
.child { width: 420px; height: 210px}
